## Sobre o projeto

O **QAPlanning** é uma plataforma desenvolvida para apoiar profissionais de tecnologia na análise de requisitos, planejamento de testes e geração de cenários utilizando Inteligência Artificial.

A aplicação auxilia QAs, Desenvolvedores e Product Owners no refinamento de histórias de usuário, identificação de riscos e criação de cenários de teste, contribuindo para entregas com maior qualidade.

**Site oficial:** https://www.qaplanning.ia.br/

> **Observação:** A documentação oficial do projeto foi mantida no **Confluence**. Este repositório tem como objetivo apresentar minha participação e as atividades desenvolvidas durante o projeto.

---

# Minha participação

Atuei como **Analista de Quality Assurance** em um projeto colaborativo, participando das atividades de planejamento, execução e validação dos testes.

Principais responsabilidades:

- Análise de requisitos
- Validação de critérios de aceite
- Elaboração de cenários utilizando Gherkin
- Execução de testes funcionais
- Registro de bugs no Jira
- Retestes após correções
- Participação na validação das histórias de usuário
- Apoio na garantia da qualidade do produto
